# Параллельное программирование - Лабораторная работа №1

## Перемножение квадратных матриц

### Задание
Разработать программу на языке C/C++ для перемножения двух квадратных матриц.

**Требования:**
- Исходные данные: файл(ы) содержащие значения исходных матриц
- Выходные данные: файл со значениями результирующей матрицы, время выполнения, объем задачи
- Автоматизированная верификация результатов вычислений с помощью сторонних библиотек или стороннего ПО

### Файлы проекта
- `lab1.cpp` - код программы
- `CMakeLists.txt` - CMake
- `matrix_a.txt` - матрица A (3×3)
- `matrix_b.txt` - матрица B (3×3)
- `verify_results.py` - скрипт для проверки результатов
